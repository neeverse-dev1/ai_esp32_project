# AI Predict Web 연동

## 1. 환경설정

> arduino ide 패키지 설치 및 python 환경구성이 필요합니다.

### (1) Arduino IDE 필요 패키지 설치

* **ArduinoJson 설치:** ArduinoJson 검색
* **NewPing 설치:** NewPing 검색 (hc_sr04 관련 라이브러리)
